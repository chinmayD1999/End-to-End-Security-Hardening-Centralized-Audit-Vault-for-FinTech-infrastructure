# End-to-End-Security-Hardening-Centralized-Audit-Vault-for-Banking-system
````````````````````````````````````````````````````````````````````````````
Designed a "Zero Trust" air-gapped Linux fintech infrastructure. Separated architecture into a Control Plane (Infra) and Workload Node (Web Client). Implemented strict security where the node requires cryptographic authorization from central infrastructure for booting, operations, and data access, ensuring total dependency on the secure core.


``````````````````````````````````````````````````````````````````````````
Identity & Access Management (IAM)
Network Hardening (Firewalld Rich Rules)
Zero-Trust Storage (LUKS + NBDE + Kerberized NFS)
Forensics & Monitoring
